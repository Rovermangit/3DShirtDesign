# 3D 衣衫设计工具

本3D衣衫设计工具原为毕业设计项目（以react库编写），内容涵盖了从网页主页、底衫选择、图片版权交易（web3.js与python图像相似度处理来实现）、后台管理系统，涉及面过大，现仅提取衣衫设计部分进行展示，通过该平台，用户可以轻松地设计出个性化的服装款式，预览 3D 效果，并进行实时调整；

## `安装使用`


### `npm install`

下载项目所需的相应依赖包；

### `npm start`

启动项目，即可在本地预览；

## `注意项`

项目所使用衣衫模型已通过Blender处理，以此实现各个部件可替换不同颜色效果；
